# password_checker
Password Strength Checker:
     This is a simple Python program that checks the strength of a password. The user enters a password, and the program counts the number of lowercase letters, uppercase letters, numbers, and special characters in it.
     Based on these counts, the program gives a password strength score and a message indicating whether the password is weak or strong.

Features :

* Accepts password input securely
* Counts different types of characters
* Calculates password strength
* Displays a strength rating and suggestion
* Easy to run from the command line

Requirements:
* Python 3

How to Run:
1. Save the program as a `.py` file.
2. Open a terminal or command prompt.
3. Run the file using Python:
   ```bash
   python filename.py
   ```
4. Enter a password when prompted.
5. View the password analysis and strength result.
